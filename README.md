신조어번역기

1. final_code.py
신조어 번역기 코드. 자체 구축한 신조어DB(data_total.xlsx)와 Open API를 사용하여 인공지능 모델을 학습시켜 이후 모델이 입력받은 신조어 포함 문장에 대해 자연스러운 번역을 하도록 하고 해당 신조어가 사용된 새 문장을 만들도록 함.

2. trans5.html
신조어 번역기와 연동되는 레이아웃 화면

3. 신조어DB
초기 사용한 자체 구축 DB. 추후 data_tital.xlsx 파일을 사용하였기 때문에 최종 신조어 번역기에는 사용되지 않음.
