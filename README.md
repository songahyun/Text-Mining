## Text-Mining: 신조어 번역기
※ This repository reorganizes my individual contributions from a team project conducted as part of a university course.  
※ 이 repository는 대학교 수업의 일환으로 진행된 팀 프로젝트에서 개인적으로 기여한 부분을 재정리한 것입니다.
___
### 프로젝트 배경:
최근 SNS와 온라인 커뮤니티를 중심으로 다양한 신조어가 빠르게 생성되면서 세대 간 언어 격차가 커지고 있다. 
이러한 신조어는 기존 사전에 등재되지 않은 경우가 많아 의미를 파악하기 어렵고, 특히 기성세대에게는 의사소통의 장벽으로 작용할 수 있다. 
따라서 신조어를 자동으로 인식하고 표준어 의미를 제공하는 시스템을 통해 신조어 이해를 돕고 원활한 의사소통을 지원할 필요가 있다.

### 프로젝트 목표:
- 문장 내 신조어를 자동으로 인식하는 번역기 개발
- 신조어의 표준어 의미 및 예문 제공
- 신조어 이해를 통한 세대 간 의사소통 지원
- 자연어 처리 기반 번역 시스템 구현
- 신조어 정보에 대한 접근성 향상

### 사용 데이터
- 네이버 오픈 사전
- 네이버 블로그, 나무위키, 디시인사이드, X(트위터) 등에서 신조어 예문 수집 → 신조어 사전 구축
___
### 프로젝트 소개: 신조어를 포함한 문자을 입력하면 해당 문장을 풀어서 자연스럽게 번역해주는 일종의 신조어 번역기.  
사용자 입력 문장에서 신조어를 여부를 확인하고 번역 및 증강한 뒤 해당 신조어를 포함하는 다른 예문과 유사한 신조어를 함께 출력.
